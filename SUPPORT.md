# Support

kruntimes is an experimental open source project maintained on a best-effort
basis.

## Questions and Usage Problems

Search the README and existing issues first. If the problem remains, open a
GitHub issue with:

- kruntimes commit or release version;
- Kubernetes and Helm versions;
- installation method and relevant configuration;
- expected and actual behavior;
- sanitized logs and resource status.

## Bugs and Features

Use the repository's bug report and feature request forms. Reproducible examples
and focused use cases make issues easier to evaluate.

## Security

Do not report vulnerabilities in a public issue. Follow
[SECURITY.md](SECURITY.md).

## Service-Level Expectations

The project does not currently provide response-time or compatibility SLAs.
Commercial support is not provided by the project.
